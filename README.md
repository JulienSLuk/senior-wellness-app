# Senior Wellness App

A web app designed for senior citizens in Singapore to improve nutrition, health awareness, and daily habits.

## Features
- Meal suggestions
- Water reminders
- Progress tracking
- Multi-language support
- Voice (text-to-speech)
- Webcam posture & activity detection

## Tech Stack
- HTML, CSS, JavaScript
- MediaPipe (pose detection)
- Web Speech API
